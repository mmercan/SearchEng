Angular Components

Service not a directive

Add "alertService" service in to your controller dependencies

angular.module("app").controller("angularcomp", ["$scope", "$http", "$routeParams", "$location","alertService", "notificationService","mainMenuService", function($scope, $http, $routeParams, $location, alertservice, notificationservice,mainMenuService) {}
NameTypeDesciption
addMoveTop function (html, options, type, autoClose) Alert on Regular colors, alertService.addMoveTop("Warning! Best check yo self, you're not looking too good.", null, "warning", 30);
addMoveTopDark function (html, options, type, autoClose) Alert on Darker colors, alertService.addMoveTopDark("Warning! Best check yo self, you're not looking too good.", null, "warning", 30);
MakeSound function (sound) produce sound for alerting user, available sounds bell, harp, error, no, notify, success, tick, warning, alien, cricket, echo, knob, alertService.MakeSound("tick");
html string html or string will shown on alert
options type: 'warning',
close_btn: true,
classes: false,
namespace: 'pa_page_alerts',
animate: true,
auto_close: false
type "warning", "error", "success","info" default is warning
autoClose optional / integer (second)
